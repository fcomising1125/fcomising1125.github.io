Concepts Learned

What is Quality Assurance?
WSSC’s Quality Assurance team’s main responsibility is to ensure that software products and processes conform to specified requirements, standards, and procedures before reaching users.
What is SDLC?
The Software Development Lifecycle (SDLC) is the process that developers use to create high-quality software. It consists of 6 quality gates:
  • Requirements Gathering
  • Analysis
  • Design
  • Implementation
  • Testing
  • Deployment
    • What is "patching?"
    Patch testing is the process of determining if a security patch caused any issues in the existing applications that they were applied to
    What are the different types of testing?
    Common testing methodologies include...
  • regression testing
  • system testing
  • integration testing
  • white-box testing
  • black-box testing
  • performance testing
  • smoke testing
  • security testing
    • What are the different types of requirements?
    Common types of requirement include...
  • system requirements: defines the technical and functional needs of the application
  • security requirements: ensures the protection and integrity of data
